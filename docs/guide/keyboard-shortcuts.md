# 键盘快捷键

## 播放控制

| 快捷键 | 操作 |
|:---|:---|
| `Space` | 播放 / 暂停 |
| `←` / `→` | 逐帧步进 |
| `J` | 上一条字幕（带预览） |
| `K` | 下一条字幕（带预览） |
| `Shift + ←` | 跳转到上一字幕 |
| `Shift + →` | 跳转到下一字幕 |

## 撤销 / 重做

| 快捷键 | 操作 |
|:---|:---|
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |

## 其他

| 快捷键 | 操作 |
|:---|:---|
| `?` | 显示快捷键帮助 |
| `Ctrl + O` | 打开视频文件 |
| `Ctrl + S` | 导出字幕 |
| `Esc` | 关闭弹窗 / 对话框 |
