# 键盘快捷键

## 导航

| 快捷键 | 操作 |
|:---|:---|
| `j` | 上一条字幕 |
| `k` | 下一条字幕 |
| `Enter` | 播放/暂停 |
| `←` | 快退 5 秒 |
| `→` | 快进 5 秒 |
| `Home` | 跳到第一条字幕 |
| `End` | 跳到最后一条字幕 |

## 编辑

| 快捷键 | 操作 |
|:---|:---|
| `E` | 编辑当前字幕 |
| `Delete` | 删除当前字幕 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |

## 视图

| 快捷键 | 操作 |
|:---|:---|
| `F` | 全屏切换 |
| `1-6` | 切换标签页（Files/Progress/ROI/OCR/Export/Settings）|

## 搜索

| 快捷键 | 操作 |
|:---|:---|
| `/` | 聚焦搜索框 |
| `Escape` | 清除搜索 / 关闭对话框 |
| `Ctrl+F` | 搜索字幕 |

## 导出

| 快捷键 | 操作 |
|:---|:---|
| `Ctrl+E` | 导出对话框 |
| `Ctrl+S` | 快速保存 |
