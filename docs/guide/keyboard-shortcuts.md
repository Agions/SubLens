# 键盘快捷键

## 全局快捷键

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `Ctrl+E` | 导出 | 打开导出面板 |
| `Ctrl+S` | 保存项目 | 保存当前项目 |
| `Ctrl+O` | 打开视频 | 打开文件对话框 |
| `Ctrl+,` | 打开设置 | 打开设置面板 |
| `F1` | 帮助 | 打开快捷键帮助 |
| `Esc` | 关闭弹窗 | 关闭模态框/面板 |

## 播放控制

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `Space` | 播放 / 暂停 | 切换播放状态 |
| `←` | 快退 5 秒 | 跳转到当前位置前 5 秒 |
| `→` | 快进 5 秒 | 跳转到当前位置后 5 秒 |
| `J` | 后退 10 秒 | JKL 导航系​​列 |
| `K` | 暂停 | JKL 导航系列 |
| `L` | 前进 10 秒 | JKL 导航系列 |
| `,` | 上一帧 | 逐帧后退 |
| `.` | 下一帧 | 逐帧前进 |
| `Home` | 跳转到开头 | |
| `End` | 跳转到结尾 | |

## 字幕编辑

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `Delete` | 删除选中字幕 | 选中字幕后删除 |
| `Ctrl+Z` | 撤销 | 撤销最近编辑 |
| `Ctrl+Shift+Z` / `Ctrl+Y` | 重做 | 恢复撤销 |
| `Ctrl+A` | 全选 | 选中所有字幕 |
| `Ctrl+D` | 复制字幕 | 复制选中字幕 |
| `Ctrl+V` | 粘贴字幕 | 粘贴到当前位置 |
| `↑` / `↓` | 选择上/下一条 | 上下切换选中 |
| `Enter` | 编辑选中字幕 | 进入编辑模式 |
| `Tab` | 接受并跳下一条 | 编辑完成后自动跳转 |

## ROI 操作

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `R` | 新建 ROI | 开始绘制新 ROI |
| `Ctrl+R` | 重置 ROI | 恢复默认预设 |
| `↑` / `↓` / `←` / `→` | 微调位置 | 每次移动 1% |
| `+` / `-` | 放大 / 缩小 ROI | 每次调整 1% |
| `Shift+拖拽` | 等比缩放 | 保持宽高比 |
| `Ctrl+拖拽` | 移动整个 ROI | 平移而不缩放 |

## OCR 与提取

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `Ctrl+Enter` | 开始提取 | 在 OCR 标签页触发 |
| `Ctrl+.` | 停止提取 | 中断正在运行的任务 |
| `Ctrl+Shift+C` | 清除结果 | 清除当前所有字幕 |

## 视图切换

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `1` | Files 标签 | 切换到文件标签 |
| `2` | ROI 标签 | 切换到 ROI 标签 |
| `3` | OCR 标签 | 切换到 OCR 标签 |
| `4` | Export 标签 | 切换到导出标签 |
| `5` | Settings 标签 | 切换到设置标签 |
| `Tab` | 下一个标签 | 循环切换 |
| `Shift+Tab` | 上一个标签 | 反向循环 |

## 批量操作

| 快捷键 | 功能 | 说明 |
|:---|:---|:---|
| `Ctrl+Shift+A` | 选中所有高置信度 | |
| `Ctrl+Shift+D` | 批量删除低置信度 | |
| `Ctrl+Shift+E` | 批量导出 | |
