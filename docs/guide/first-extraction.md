# 首次提取操作流程

## 操作步骤概览

```
导入视频 → 选择 ROI → 选择引擎 → 开始提取 → 审核结果 → 导出字幕
```

## Step 1 — 导入视频

1. 点击工具栏 **「导入视频」** 按钮，或直接拖拽视频文件到窗口
2. 支持格式：MP4 · MKV · AVI · MOV · WebM · M4V · WMV · FLV · 3GP
3. 视频加载后，会自动提取第一帧显示在预览区

## Step 2 — 选择 ROI 区域

ROI（Region of Interest）定义了字幕在画面中的位置。

### 预设区域

| 预设 | 适用场景 |
|:---|:---|
| **底部** | 最常见的硬字幕位置（电影、综艺） |
| **顶部** | OP/ED 字幕、新闻滚动条 |
| **左侧** | 翻译类字幕（左侧原文，右侧译文） |
| **右侧** | 双语字幕的其中一侧 |
| **中间** | 居中对话字幕 |
| **自定义** | 拖拽选择任意矩形区域 |

### 自定义 ROI

1. 在视频预览区拖拽鼠标绘制矩形
2. 支持四角拖拽调整大小
3. 可输入精确的百分比坐标

> **建议：** 先用「自动检测」功能扫描全帧，找出字幕出现频率最高的区域，再手动微调。

## Step 3 — 选择 OCR 引擎

| 引擎 | 精度 | 速度 | 语言数 | 显存需求 |
|:---|:---:|:---:|:---:|:---:|
| **PaddleOCR** | ★★★ | 快（GPU） | 80+ | ~2GB GPU |
| **EasyOCR** | ★★★ | 中等 | 80+ | ~4GB GPU |
| **Tesseract.js** | ★★ | 快（CPU） | 100+ | 无 |

初次使用推荐 **Tesseract.js**（零配置，即装即用）。

## Step 4 — 配置高级选项（可选）

展开「高级选项」面板：

- **多轮 OCR**：同一区域识别多次，取最优结果（精度↑，速度↓）
- **文本后处理**：全角/半角标点规范化
- **字幕合并**：Levenshtein 相似度智能去重
- **场景检测灵敏度**：跳过无字幕帧

## Step 5 — 开始提取

1. 点击 **「开始提取」**
2. 进度条实时显示：已处理帧数 / 总帧数
3. 场景检测完成后自动进入 OCR 阶段
4. 提取结果实时显示在字幕列表中

### 提取过程说明

```
[场景检测] ████████████ 100%  检测到 48 个场景
[OCR 识别] ██████████░░░░░░  67%  156 / 233 帧
```

## Step 6 — 审核与编辑

提取完成后，在字幕列表中可以：

- **逐条审核**：点击任意条目，播放器跳转对应时间点
- **修改文本**：双击单元格直接编辑
- **删除条目**：选中后按 `Delete` 或点击删除按钮
- **过滤低置信度**：点击「低置信度」标签，只显示需要审核的条目

### 置信度等级

| 等级 | 颜色 | 置信度范围 |
|:---|:---|:---|
| 高 | 绿色 | ≥ 0.85 |
| 中 | 黄色 | 0.60 ~ 0.85 |
| 低 | 红色 | < 0.60 |

## Step 7 — 导出

1. 切换到 **「导出」** 标签页
2. 选择目标格式（默认 SRT）
3. （可选）勾选「包含时间码」「包含置信度」
4. 点击 **「导出」** 按钮
5. 选择保存路径

## 快捷键

| 快捷键 | 功能 |
|:---|:---|
| `Space` | 播放 / 暂停 |
| `←` / `→` | 快退 / 快进 5 秒 |
| `J` / `K` / `L` | 后退 / 暂停 / 前进（JKL 导航）|
| `Delete` | 删除选中字幕 |
| `Ctrl+E` | 导出 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+S` | 保存项目 |
