# 首次提取

本指南带你完成从安装到导出第一个字幕文件的完整流程。

## 步骤概览

1. 打开视频文件
2. 选择字幕区域（ROI）
3. 配置 OCR 参数
4. 开始提取
5. 审核与编辑
6. 导出字幕

---

## 第一步 — 打开视频

点击工具栏 **Open** 按钮，或直接将视频文件拖入窗口。

支持的格式：**MP4 · MKV · AVI · MOV · WebM · M4V · WMV · FLV · 3GP**

---

## 第二步 — 选择字幕区域

SubLens 默认使用 **Bottom**（底部 20%）预设，适合大多数硬字幕场景。

如需调整：

- **拖拽** ROI 边框调整大小
- **拖拽** ROI 内部调整位置
- 或从预设列表选择其他模式

---

## 第三步 — 配置 OCR

### 选择引擎

| 引擎 | 特点 |
|:---|:---|
| EasyOCR | 推荐首选，精度与速度平衡 |
| PaddleOCR | GPU 用户极限精度 |
| Tesseract.js | 无 Python 环境快速上手 |

### 选择语言

选择视频中字幕使用的语言。多语字幕可多选。

### 推荐设置

| 设置 | 推荐值 |
|:---|:---|
| 置信度阈值 | 70% |
| 多轮 OCR | 启用 |
| 文本后处理 | 启用 |
| 字幕合并 | 启用（相似度 80%）|

---

## 第四步 — 开始提取

点击 **Start Extraction** 按钮。

进度显示在 **Progress** 标签页：

- 当前处理的帧
- 已识别字幕数量
- 预计剩余时间

提取完成后自动跳转到字幕列表。

---

## 第五步 — 审核与编辑

提取结果可能需要少量人工审核：

- **低置信度字幕**（红色标记）建议重点检查
- 点击字幕跳转到对应视频时间点
- 使用 `j` / `k` 键快速浏览
- 双击字幕可直接编辑文本

---

## 第六步 — 导出

1. 点击字幕面板的 **Export** 按钮
2. 选择导出格式：

| 用途 | 推荐格式 |
|------|---------|
| 通用播放器 | SRT |
| 帧级精确编辑 | JSON |
| 动漫字幕 | ASS |
| 数据分析 | CSV |

3. 选择保存位置
4. 点击 **Export**
