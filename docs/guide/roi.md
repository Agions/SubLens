# ROI 区域选择

ROI（Region of Interest，感兴趣区域）定义视频中字幕出现的位置，是提取准确率的关键。

## 预设模式

| 预设 | 说明 | 适用场景 |
|:---|:---|:---|
| **Bottom** | 底部 20% 区域 | 大多数硬字幕 |
| **Top** | 顶部 20% 区域 | 片头/片尾字幕 |
| **Left** | 左侧 50% 区域 | 左侧双语字幕 |
| **Right** | 右侧 50% 区域 | 右侧双语字幕 |
| **Center** | 中间 60% 区域 | 对话叠加字幕 |
| **Custom** | 自由拖拽定义 | 复杂布局 |

## 自定义 ROI

1. 选择 **Custom** 预设
2. 在视频预览区拖拽绘制矩形
3. 支持四角拖拽调整大小
4. 支持拖拽移动位置

## ROI 坐标系统

ROI 使用百分比坐标系（相对于视频分辨率）：

```typescript
interface ROI {
  x: number      // 左上角 x（0-100%）
  y: number      // 左上角 y（0-100%）
  width: number  // 宽度（0-100%）
  height: number // 高度（0-100%）
}
```

百分比坐标系确保在不同分辨率视频间正确缩放。

## 多字幕区域

对于同时有多个字幕区域（如双语字幕），可以：

1. 先提取第一个区域（Bottom）
2. 调整 ROI 到第二个区域（Top）
3. 合并两次提取结果
