# ROI 区域选择

## 什么是 ROI

ROI（Region of Interest，兴趣区域）定义了字幕在视频画面中出现的范围。

精确的 ROI 可以：
- **减少无效 OCR**：跳过画面中不含字幕的区域
- **提升识别速度**：只处理字幕区域，帧处理速度提升 3-10×
- **降低误识别**：避免将 LOGO、水印、装饰文字识别为字幕

## 预设区域

| 预设 | 坐标（相对值）| 适用场景 |
|:---|:---|:---|
| 底部 | `y: 0.75~1.0` | 电影、综艺、纪录片硬字幕 |
| 顶部 | `y: 0.0~0.25` | OP/ED 字幕、新闻滚动条 |
| 左侧 | `x: 0.0~0.5, y: 0.7~1.0` | 翻译类字幕（左侧原文）|
| 右侧 | `x: 0.5~1.0, y: 0.7~1.0` | 双语字幕的其中一侧 |
| 中间 | `x: 0.2~0.8, y: 0.8~1.0` | 居中对话字幕 |
| 全屏 | `x: 0.0~1.0, y: 0.0~1.0` | 测试用，不推荐日常使用 |

## 自定义 ROI

### 界面操作

1. 在视频预览区，单击并拖拽定义矩形区域
2. 选中后可拖拽四边调整大小
3. 双击进入精确数值编辑模式

### 坐标说明

ROI 使用**相对坐标**（0.0 ~ 1.0），适配任意分辨率视频：

```
┌─────────────────────────────────┐  y=0.0
│                                 │
│          x=0.0                  │
│    ┌──────────────────┐         │
│    │   ROI 区域       │         │  y=0.75
│    └──────────────────┘         │
│                        x=1.0    │
└─────────────────────────────────┘  y=1.0
```

### 百分比 vs 像素

| 模式 | 优点 | 缺点 |
|:---|:---|:---|
| **百分比** | 自适应任意分辨率 | 无法精确控制像素级细节 |
| **像素** | 精确控制 | 换分辨率后需重新调整 |

SubLens 内部统一使用百分比存储，导出/导入时自动转换。

## ROI 预设管理

### 保存自定义预设

在 ROI 编辑完成后，点击「保存为预设」，输入预设名称。

预设保存在 `~/.sublens/roi-presets.json`。

### 批量应用

批量处理时，可以：
- 为不同视频分别设置不同 ROI
- 从已有视频导入 ROI 配置
- 导出 ROI 配置为 JSON 复用

## 自动 ROI 检测

点击「自动检测」，SubLens 会：

1. 均匀采样 30 帧
2. 对每帧进行全屏 OCR
3. 统计文字出现的 y 坐标分布
4. 自动选中文字密度最高的区域

> **注意**：自动检测仅作为初值，最终仍需人工审核调整。

## ROI 与字幕合并

ROI 和 Pipeline 的 `mergeSimilar` 阶段配合工作：

- 同一 ROI 内的字幕：正常合并去重
- 不同 ROI 的字幕（如顶部+底部）：视为独立流，不合并

## 快捷操作

| 操作 | 快捷键 |
|:---|:---|
| 新建 ROI | `R` |
| 重置 ROI | `Ctrl+R` |
| 上移 1% | `↑` |
| 下移 1% | `↓` |
| 左移 1% | `←` |
| 右移 1% | `→` |
| 放大 1% | `+` |
| 缩小 1% | `-` |
